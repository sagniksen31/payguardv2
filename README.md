# 🛡️ PayGuard — ATM Intelligence & Automation System

PayGuard is an AI-powered system designed to **predict ATM failures, identify root causes, and automate issue resolution** to reduce downtime and improve service reliability.

---

## 🚀 Overview

Financial infrastructure like ATMs suffers from:
- Frequent downtime
- Delayed issue resolution
- High manual intervention

PayGuard solves this by combining:
- Predictive AI models
- Root Cause Analysis (RCA)
- Automation Engine

---

## 🎯 Key Features

### 🔮 Predictive Failure Detection
- Risk Score (0–100)
- Risk Label (LOW / MEDIUM / HIGH)
- Escalation Probability

---

### ⚡ Real-Time System State
- Displays current health of ~100 ATMs
- Risk categorization:
  - High Risk
  - Medium Risk
  - Low Risk

---

### 🤖 Automation Engine
- AUTO_RESOLVED
- AUTO_ATTEMPTED
- MANUAL_REQUIRED

Metrics:
- Auto resolved incidents
- Downtime prevented
- Manual effort saved

---

### 🔍 Root Cause Analysis
- Repeat failures
- Issue clusters
- Cascade failures

---

### ⚡ Live Prediction
- Instant risk prediction

---

### 📂 Data Input Modes
- Generate Demo Data
- Upload CSV
- Manual Entry
